Git and Github commands


SETUP

Git --version
Git config --global user.name ["user name" over Github profile]
Git config --global user.email ["user email" over Github login]

CLONE & STATUS

Git clone [HTTPs link]
Git status

ADD & COMMIT

Git add [file name]
Git commit -m ["some message"]

PUSH COMMAND

Git push origin main [branch name]

INIT COMMAND

Git init
Git remote add origin [Http link]
Git remote -v[To verify remote]
Git branch
Git branch -M [branch name] like main
Git push -u origin main [-u is use when we work on same project
Git push

CREATE NEW BRANCH

Git branch
Git checkout -b [New branch name]
Git checkout [branch name]
